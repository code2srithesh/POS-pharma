## About

POS-pharma is intended to support the core operations of a retail pharmacy:
- Track products, batches, expiry dates and stock movement
- Create and process sales (invoices/receipts)
- Handle prescriptions and controlled substances workflows
- Manage customers and suppliers
- Generate sales, inventory and tax reports
- User roles and basic access control (e.g., cashier, pharmacist, manager)
